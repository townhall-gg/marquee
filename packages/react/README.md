# @townhall-gg/marquee-react

React bindings for the marquee animation library.

See the [main README](https://github.com/townhall-gg/marquee) for full documentation.

## Installation

```bash
pnpm add @townhall-gg/marquee-react
```

## Usage

```tsx
import { Marquee } from "@townhall-gg/marquee-react";

function App() {
  return (
    <Marquee speed={100}>
      <span>Your scrolling content&nbsp;</span>
    </Marquee>
  );
}
```
