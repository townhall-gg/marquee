# @townhall-gg/marquee

Core marquee animation library using the Web Animations API.

See the [main README](https://github.com/townhall-gg/marquee) for full documentation.

## Installation

```bash
pnpm add @townhall-gg/marquee
```

## Usage

```ts
import { createMarquee } from "@townhall-gg/marquee";

const marquee = createMarquee(element, { speed: 100 });
marquee.play();
```
